# Slice 73 Evaluation: Platform Fee Memory and RPG Homestead Profiles

## Goal

Capture the latest visionary direction: CryptoSeeds is a self-custodial third-party platform/interface where users hold their own assets, project owners and charities can be discovered through the MicroVerse, fees are transparent, and higher RYP tiers create larger customizable homesteads.

## Added

- Homestead customization domain model by staking tier.
- Unit tests proving homestead scale and customization capacity increase by tier.
- Homestead profile surfaced in the Homestead dashboard metric row.
- Project Fields header now shows field count and decoration capacity.
- Master brief update for the remembered fee structure and SeedBot fee concept.
- State-model and visual-bible updates for epic RPG-style homestead customization.

## Fee Memory

- Base network/platform fee: 3.5%.
- Tier reductions:
  - Seed: 3.5%.
  - Sprout: 3.15%.
  - Sapling: 2.8%.
  - Tree: 2.45%.
  - Fruit: 2.1%.
- Network fee categories: holders, stakers, independent treasury.
- Exact holder/staker/treasury percentages are still configurable.
- SeedBot success-fee concept: 12% of realized positive strategy PnL only, split 40% dev / 60% treasury, review-gated and disabled for live use until security/legal review.

## Product Call

The public posture should be: self-custodial platform, wallet-approved actions, project discovery, project-owner/charity access, transparent fees, and user-controlled strategy tools.

Do not publicly say that the platform avoids law or cannot be regulated. Keep the build decentralized and non-custodial, then get the final public positioning reviewed before launch.
